# Core Configuration

| Name           | Type           | Default | Required? | Description                        | Example                 |
|----------------|----------------|---------|-----------|------------------------------------|-------------------------|
| `measure_year` | `Number`       | null    | Yes       | The measure year.                  | `2020`                  |
| `measures`       | `List[String]` | null    | Yes       | A list of the measure keys to run. | `['aba', 'bcs', 'trc']` |
